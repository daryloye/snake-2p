# Snake 2P

A simple two-player Snake game playable in the terminal.

## Requirements
- Linux
- C++ compiler (`g++`)
- `ncurses` library

## Installation

```bash
git clone https://github.com/daryloye/snake_2p.git
cd snake_2p
make
./snake
```

## Controls

### Player 1
- `A`: Turn left
- `D`: Turn right

### Player 2
- `Left Arrow`: Turn left
- `Right Arrow`: Turn right
